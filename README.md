# UFO Sightings Dashboard

### Instalação

- Utilizar Python 3.
- Colocar todos os arquivos na mesma pasta.

No terminal ou command line, executar os seguintes comandos:
```sh
$ pip install dash
$ pip install pandas
$ pip install plotly
$ pip install dash-d3cloud
```
Uma vez que todas as bibliotecas estejam instaladas, abrir o terminal ou command line e acessar a pasta em que os arquivos estão salvos. Para executar o Dashboard, execute o seguinte comando:
```sh
$ python app.py
```
Acesse o endereço de localhost (por exemplo: http://127.0.0.1:8050/).
